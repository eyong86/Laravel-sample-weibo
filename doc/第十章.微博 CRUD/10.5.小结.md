最后，我们需要将代码进行提交，并切换到主分支上进行合并。

```bash
$ git add -A
$ git commit -m "Add user statuses"
$ git checkout master
$ git merge user-statuses
```

接着将代码推送到 GitHub 和 Heroku 上：

```bash
$ git push
$ git push heroku master
```

由于我们新增了微博迁移文件，因此需要在 Heroku 上也需要执行迁移。

```bash
$ heroku run php artisan migrate
```
